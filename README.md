# Quiz Application 🎯

## 🌟 Giới thiệu
Dự án **Quiz App** là một ứng dụng web trắc nghiệm tương tác, cho phép người dùng tham gia trả lời các câu hỏi, kiểm tra kiến thức và nhận kết quả tổng kết ngay lập tức. Dự án này được xây dựng nhằm mục đích tối ưu hóa kỹ năng xử lý logic frontend, quản lý trạng thái (state) và tương tác với người dùng theo thời gian thực.

## ✨ Tính năng nổi bật
* **Ngân hàng câu hỏi:** Hiển thị câu hỏi động từ tập dữ liệu có sẵn.
* **Tương tác trực tiếp:** Lựa chọn đáp án trắc nghiệm (Multiple-choice) mượt mà.
* **Tính điểm tự động:** Chấm điểm và hiển thị kết quả chi tiết, thống kê số câu đúng/sai sau khi hoàn thành bài thi.

## 🛠 Công nghệ sử dụng
* **Core:** HTML5, CSS3 / SCSS, JavaScript (ES6+)
* **Framework / Build Tool:** ReactJS*
* **Database / API:** JSON Server.
